# Flipkart Customer Service Satisfaction Prediction

## Project Overview
This project builds a machine learning classification model to predict customer satisfaction based on customer support interaction data. By analyzing factors such as response time, issue category, agent interaction, and communication channel, the project aims to identify key drivers of customer satisfaction and provide insights to improve service quality.

## Objectives
- Analyze customer support interaction data
- Identify factors influencing customer satisfaction
- Build a machine learning classification model
- Generate insights to improve customer service operations

## Dataset
The dataset contains customer support interaction records including:
- Response Time
- Issue Category
- Communication Channel
- Agent Interaction Details
- Customer Satisfaction Label

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Data Visualization

## Project Workflow
1. Data Collection
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Training and Evaluation
6. Insights and Interpretation

## Results
The model identifies important factors influencing customer satisfaction such as response time, issue resolution efficiency, and interaction quality.
